# IT Support Diagnostic System

A Node.js-based IT support diagnostic tool that helps automate troubleshooting workflows and issue escalation.

## Overview

The IT Support Diagnostic System is a command-line application designed to simulate common IT support troubleshooting procedures. It helps identify and respond to frequently reported technical issues while escalating unknown problems to Level 2 support.

## Features

* Diagnose common IT support issues
* Automated troubleshooting workflows
* Issue escalation for unrecognized problems
* Interactive command-line interface
* Built with JavaScript and Node.js

## Supported Issues

The system currently supports:

* No Internet
* Slow PC
* Printer Offline
* Email Not Syncing
* Blue Screen Errors

## Technologies

* JavaScript
* Node.js
* Readline Module

## How to Run

1. Install Node.js
2. Clone the repository
3. Open a terminal in the project directory
4. Run:

```bash
node it-support-diagnostic-system.js
```
